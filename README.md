## Features

| Name | Description |
|------|-------------|
| Real-Time Tax Calculation | Instant sales tax, VAT, and GST calculations for every transaction across all jurisdictions globally. |
| Exemption Certificate Management | Automated collection, validation, and storage of tax exemption certificates through CertCapture. |
| E-Invoicing Compliance | Country-specific e-invoicing mandate support for cross-border and domestic compliance workflows. |
| Address Validation | Postal address validation and tax jurisdiction determination for US and international addresses. |
| HS Code Classification | AI-powered Harmonized System Code and Avalara Tax Code classification for products. |
| MCP Server Support | Model Context Protocol servers connecting AI applications with Avalara tax compliance systems. |
| OData Query Support | AvaTax REST API supports OData query parameters including filter, top, skip, and orderBy. |
| Sandbox Environment | Dedicated sandbox environment at sandbox-rest.avatax.com for testing without production impact. |
